# Crear Matriz en C#

Este proyecto es una aplicación de consola desarrollada en **C#** utilizando Visual Studio.

## Descripción

El programa:

1. Solicita al usuario el número de filas y columnas.
2. Permite ingresar los valores de una matriz.
3. Evalúa cada número:

   * Si es **par**, lo multiplica por 2.
   * Si es **impar**, lo multiplica por 3.
4. Muestra la nueva matriz resultante en pantalla.

## Tecnologías utilizadas

* C#
* .NET
* Visual Studio

## Ejemplo de funcionamiento

Entrada:

```
2 3 
1 2 3
4 5 6
```

Salida:

```
3 4 9
8 15 12
```
